# AI for Medical Diagnostics

A personal research wiki on AI for medical diagnostics, built using the LLM-wiki pattern: an LLM owns and maintains the wiki pages; the human curates sources and asks questions.

## Start here

- **[Overview](wiki/overview.md)** — evolving top-level synthesis of the field.
- **[Index](wiki/index.md)** — content catalog; every page listed with a one-line summary.

## Layout

- `raw/` — immutable source documents (PDFs, articles, transcripts). Read-only.
- `wiki/` — LLM-maintained markdown pages (sources, entities, concepts, comparisons).
- `CLAUDE.md` — the schema and workflows (ingest, query, lint).

See `CLAUDE.md` for conventions and the full schema.
